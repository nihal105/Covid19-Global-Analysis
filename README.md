# Covid19-Global-Analysis
## About the Project

This repository contains a Power BI dashboard project focused on analyzing global COVID-19 data.
The dashboard converts complex raw data into meaningful insights through interactive visuals, KPIs, and trend analysis.

The project emphasizes:

Data accuracy

Clean dashboard design

Beginner-to-intermediate Power BI techniques

## Key Insights Delivered

Total confirmed, active, recovered, and death cases

COVID-19 trends over time

Country-wise and region-wise comparison

Global impact visualization using maps

## Tech Stack
Tool	Purpose
Power BI	Dashboard & visualization
Power Query	Data transformation
DAX	Calculations & KPIs

## Dashboard Components

KPI Cards – Quick snapshot of key numbers

Line Charts – Trend analysis over time

Bar Charts – Top affected countries

Map Visuals – Geographic spread

Slicers – Date, country, and region filters

Bookmarks – Enhanced navigation experience
